## REQUIREMENT
[INSTALL NPM & NODE](https://nodejs.org/en/download/)

## DESCRIPTION
`Search info on www.tiktok.com`

## SETUP
#### - NODE
```bash
      $ npm i puppeteer
```
#### - INDEX.JS
Modified this line in your code
```
var tagSearch = "yourSearchTagName";
var fileJson = "tiktok.json";
```

## RUN
```bash
      $ npm start
```
